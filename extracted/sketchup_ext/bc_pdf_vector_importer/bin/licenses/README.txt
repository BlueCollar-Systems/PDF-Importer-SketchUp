Bundled Poppler helper license notes

The SketchUp PDF Vector Importer Windows RBZ includes Poppler command-line
utilities and the DLLs required by the poppler-windows build.

Upstream project:
  https://github.com/oschwartz10612/poppler-windows

Poppler itself is GPL-family licensed. The Windows bundle also includes
companion libraries under their own upstream licenses. When refreshing these
binaries with tools/fetch_third_party_binaries.ps1, keep any upstream COPYING,
LICENSE, NOTICE, README, AUTHORS, or COPYRIGHT files copied into this folder.
