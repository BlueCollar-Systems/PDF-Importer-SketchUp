# BlueCollar Structural Steel SU Shapes

Free SketchUp shape pack generated from AISC v16.0 structural steel data.

> **Repository location:** this pack now lives in
> [PDF-Importer-SketchUp](https://github.com/BlueCollar-Systems/PDF-Importer-SketchUp)
> under `steel_shapes/`. See [ATTRIBUTION.md](ATTRIBUTION.md) for merge history.

## Folder Layout

| Path | Description |
|------|-------------|
| `family_packs/` | 14 `.skp` family files |
| `CHECKSUMS.md` | SHA-256 values for release ZIPs |
| `LICENSE` | CC0-1.0 |

- Browse `family_packs/` for individual `.skp` files.
- Or download a versioned ZIP from
  [PDF-Importer-SketchUp Releases](https://github.com/BlueCollar-Systems/PDF-Importer-SketchUp/releases)
  (assets named `Structural-Steel-SU-Shapes-*.zip`, tagged `steel-v*`).
